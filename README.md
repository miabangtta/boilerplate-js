# Boilerplate for js projects

## Workspace
* ES6 support (babel)
* webpack 4
* stylus
* dev-server

## Usage

### Installation

Install the dependencies

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start
```

### Build
Compile and build

```sh
$ npm run build
```
## Navigation:
* Main page: https://localhost:8080
